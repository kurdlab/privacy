What information I collect and how I use them

The application uses third party services that collect data that may identify you. The application uses aggregated and non-personalized data provided by third party services to

  • analyse and understand user behavior

  • test Application functionality

  • make Application work as well as possible

When you turn ads on to get the all features for free, Application starts using Google AdMob and InMobi services to mediate and serve ads. The services use data to target ads.

Third party services used by Application and link to their privacy policies:

  • Google services: https://policies.google.com/privacy

  • Google analytics for Firebase: https://firebase.google.com/policies/analytics/. The application collects anonymous and aggregated data:

    • whether ads are turned on

    • windows opened in application (like calculation details, theme window, help etc.)

    • pressed keys on calculator keyboard

  • InMobi: https://www.inmobi.com/privacy-policy/ (only when ads are turned on) - geo-location and WiFi data are not collected

When the application crashes, the user is asked to send a crash report. If the user provides consent, the email client is opened and prefilled with stack trace, calculator settings, device information and activity information before the crash. All data are in a text form. The user can review them and decide wheter to send them or not.
